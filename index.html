Welcome to BioJSON community site.

The BioJSON project aims at introducing a standardise JavaScript Object Notation (JSON) format for exchange of Bioinformatics Sequence Alignment data.

JSON is a lightweight data-interchange format, it is language independent, "self-describing" and easy to understand. JSON uses JavaScript syntax, but the JSON format is text only. Hence, it can be read and used as a data format by any programming language. More information about JSON can be found at http://json.org/

The key benefits of BioJson will be its ability to represent any Sequence Alignment data, along with its meta-data and annotations in a structured manner.

In the future API's will be created for transforming any Bioinformatics data format to a BioJson format.
